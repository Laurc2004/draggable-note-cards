## 简介

✨ **Draggable Note Cards** 是一个简单易用的拖拽笔记卡片工具，支持自由拖拽、实时编辑功能。

## 当前功能

- 📍 拖拽调整卡片位置：自由移动卡片，调整布局随心所欲
- ✏️ 实时编辑卡片内容：随时修改卡片内容，实时保存
- ➕ 添加和删除卡片内容：轻松添加新卡片或删除不需要的卡片
- 💾 本地持久化存储卡片内容：卡片内容保存在本地，刷新后也不会丢失
